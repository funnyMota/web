# 常见样式声明

## `color`

- 元素内部文字颜色
- **预设值**：定义好的单词
- **三原色/色值**：`rgb(红,绿,蓝)`，每个颜色由 0-255 的数字表达

```css
color: rgb(140, 0, 255);
/* hex（16进制）表示法：#RRGGBB 或 #RGB */
color: #ff4400;
```

### 常见颜色

- 淘宝红：`#ff4400` / `#f40`
- 黑色：`#000000` / `#000`
- 白色：`#ffffff` / `#fff`
- 红：`#ff0000` / `#f00`
- 绿：`#00ff00` / `#0f0`
- 蓝：`#0000ff` / `#00f`

## `background-color`

- 元素背景颜色

## `font-size`

- 元素内部文字尺寸
- `px`：像素，可简单理解为文字高度占多少像素
- `em`：相对单位，相对于父元素的字体大小

每个元素必须要有字体大小：如果没有声明，则使用父元素的字体大小；如果没有父元素，则使用基准字号。

> `strong` 默认加粗
> user agent（UA）：用户代理（浏览器）

## `font-weight`

- 文字粗细，可取数字或预设值

## `font-family`

- 字体簇
- 必须用户电脑中存在该字体才会生效
- 常在末尾加上 `sans-serif` 作为兜底字体族

## `font-style`

- 字体样式，常用于斜体

> `i` 默认样式是倾斜字体
> `em` 默认是斜体

## `text-decoration`

- 文本修饰（加线等）

> `a` 默认带下划线
> `del` 默认带删除线
> `s` 表示过期内容

## `text-indent`

- 首行缩进

## `line-height`

- 行高（每行文本高度）
- 行高设置为容器高度可让单行文本垂直居中
- 行高可设置为纯数字，表示相对于当前元素字体大小的倍数

## `width` / `height`

- `width`：宽度
- `height`：高度

## `letter-spacing`

- 字间距

## `text-align`

- 元素内部文字的水平对齐方式
