# 选择器

选择器：精准选中想要的元素。

## 简单选择器

1. **ID 选择器**
2. **元素选择器**
3. **类选择器**：`class`
4. **通配符选择器**：`*`
5. **属性选择器**：`[]`（根据属性名和属性值选择元素）
6. **伪类选择器**：选中某些元素的某种状态
   - `:link`：超链接未访问时
   - `:visited`：超链接访问过
   - `:hover`：鼠标悬停
   - `:active`：鼠标按下

   爱恨法则：`link` → `visited` → `hover` → `active`
7. **伪元素选择器**
   - `::before`
   - `::after`

## 选择器的组合

1. 并且：，
2. 后代元素：空格
3. 子元素：`>`
4. 相邻兄弟元素：`+`
5. 兄弟元素：`~`

## 选择器的并列

多个选择器用逗号分隔。
