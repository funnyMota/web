# a元素

## href属性
`href`：reference（引用），表示跳转目标。

1. 跳转某个地址
2. 跳转某个锚点
3. 功能链接：点击后触发某个功能

`id`：全局属性，表示元素在文档的唯一编号。

### 常见功能链接

- `href="mailto:33070@qq.com"`：发送邮件  
  需要用户电脑安装邮件客户端（如 Outlook / Exchange 等）

- `href="tel:33070"`：拨号  
  需要系统/浏览器具备拨号能力

## target属性
`target`：表示跳转打开的位置。

- _self:在当前页面窗口打开
- _blank:在新的标签页打开
