# 图片元素

## `img` 元素

- `img`：image，空元素
- `src`：source 的缩写
- `alt`：当图片资源失效时，用该文字替代图片
- `usemap`：使用图片地图

## `map` 元素

- `map`：地图
- 子元素：`area`
- `shape`：`rect` / `circle` / `poly`
- `rect`：`coords="x1,y1,x2,y2"`
- `circle`：`coords="x,y,r"`
- `poly`：`coords="x1,y1,x2,y2,...,xn,yn"`

## figure元素

`figure`：指代定义，通常把图片及其说明信息包裹起来。
子元素：`figcaption`
